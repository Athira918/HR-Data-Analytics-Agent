# HR-Data-Analytics-Agent

An AI-powered HR Data Analytics Agent built using Activepieces to automate HR data processing, analysis, visualization, and reporting. This workflow transforms raw HR datasets into actionable insights by performing data cleaning, metric calculations, chart generation, and automated report delivery.

Project Overview

Managing HR data manually can be time-consuming and error-prone. This project automates the entire analytics pipeline, enabling HR teams to quickly understand workforce trends and make data-driven decisions.

The agent accepts HR datasets, cleans and processes the data, generates key insights and visualizations, creates an HTML analytics report, and automatically sends the report via email.

Features

Automated HR Data Processing
Reads HR datasets automatically.
Extracts relevant employee information.
Handles missing values and inconsistent data.
Data Cleaning and Transformation
Removes duplicates and invalid records.
Standardizes columns and formats.
Prepares data for analysis.
HR Analytics Metrics

Generates important HR KPIs such as:

Total number of employees
Average age of employees
Gender distribution
Department-wise employee count
Job role distribution
Education level analysis
Marital status distribution
Employee attrition count
Average monthly income
Work-life balance statistics
Dynamic Data Visualization

Automatically creates charts for:

Department-wise employee distribution
Gender distribution
Attrition analysis
Education field distribution
Job role analysis
Monthly income trends
Automated HTML Report Generation

Creates a structured analytics report containing:

Summary statistics
Key HR insights
Interactive visualizations
Workforce trends and patterns
Email Automation

Automatically delivers the generated report via email, reducing manual reporting effort.
